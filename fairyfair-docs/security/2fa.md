# Двухфакторная аутентификация (2FA)

## Обязательность

2FA является **обязательной** для всех пользователей `admin.fairyfair.ru` — без возможности отключения.

При первом входе пользователь принудительно проходит настройку TOTP перед получением доступа к любому модулю.

---

## Стандарт: TOTP

Используется стандарт **TOTP (Time-based One-Time Password)**, совместимый с:

- Google Authenticator
- Яндекс.Ключ
- Любым другим TOTP-совместимым приложением

---

## Хранение TOTP-секретов

TOTP-секреты хранятся в БД в **зашифрованном виде** — алгоритм **Fernet (AES-128-CBC)**.

Ключ шифрования секретов хранится отдельно от ключей шифрования остальных данных. Это означает: при компрометации основной базы данных секреты 2FA остаются защищены отдельным ключом.

```
БД: [зашифрованный TOTP-секрет]
                ↓
   расшифровка ключом 2FA (хранится отдельно)
                ↓
         TOTP-секрет (в памяти)
```

---

## Сброс 2FA

При утере TOTP-устройства пользователь обращается к IT-администратору. Сброс производится через CRM → «Пользователи» → карточка сотрудника.
