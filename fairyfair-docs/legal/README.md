# Юридические документы

Перечень документов, обязательных к размещению согласно законодательству РФ.

---

## Публичный сайт (fairyfair.ru)

| Документ | Назначение |
|----------|-----------|
| **Пользовательское соглашение** | Условия использования сайта и сервисов платформы |
| **Политика конфиденциальности** | Порядок сбора, хранения и обработки персональных данных (152-ФЗ) |
| **Публичная оферта (билеты)** | Условия продажи билетов, права и обязанности сторон |
| **Политика возврата** | Условия и сроки возврата билетов |
| **Правила мероприятия** | Правила поведения на территории фестиваля |
| **Согласие на обработку персональных данных** | Чекбокс при регистрации и покупке билета (152-ФЗ) |
| **Согласие на съёмку** | Уведомление на входе или в правилах мероприятия |

> Согласие на обработку персональных данных должно быть получено явным образом — чекбокс без предустановленной галочки.

## Соответствие законодательству

### 152-ФЗ «О персональных данных»

- Персональные данные пользователей хранятся на серверах, расположенных на территории РФ
- Логирование действий пользователей в течение установленного законом периода
- Техническая защита персональных данных (шифрование, контроль доступа)
- Явное согласие пользователя на обработку ПД при регистрации и покупке

### 115-ФЗ «О противодействии легализации доходов»

- Приём оплаты осуществляется **только через легальные платёжные шлюзы** (ЮКасса, Тинькофф и др.)
- Переводы с карты на карту для приёма оплаты **не используются**
- Все финансовые операции проходят через подключённый платёжный шлюз с формированием фискальных документов
