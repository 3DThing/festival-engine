# staff.fairyfair.ru — Панель участников

Поддомен для участников мероприятия, не входящих в команду организаторов. Доступ открывается после регистрации и прохождения аккредитации.

## Категории участников

### Торговцы

Поля при регистрации:

| Поле | Описание |
|------|----------|
| Статус | Самозанятый / ИП / ООО |
| ФИО представителя | Контактное лицо |
| Название торговой точки | Публичное название |
| ФИО команды | Состав (при наличии) |
| Категория товаров | Тип продукции |

### Музыканты

Поля при регистрации:

| Поле | Описание |
|------|----------|
| Название группы / исполнителя | Публичное название |
| Состав | Участники коллектива |
| Жанр | Музыкальное направление |
| Фото / обложка | Изображение для программы |

### Косплееры

Поля при регистрации:

| Поле | Описание |
|------|----------|
| ФИО | Полное имя участника |
| Персонаж | Имя персонажа |
| Фото | Фото косплея |
| Референс персонажа | Изображение оригинала |

### Фотографы

Поля при регистрации:

| Поле | Описание |
|------|----------|
| ФИО | Полное имя |
| Портфолио | Ссылка или загрузка |
| Оборудование | Описание техники |
| Согласие | Условия передачи фото организаторам |

---

## Функционал панели

### Аккредитация

1. Участник заполняет форму согласно своей категории
2. Принимает соответствующую оферту
3. Заявка уходит на рассмотрение организаторам
4. После одобрения — выдаётся QR-код аккредитации

Статусы заявки: `Ожидает` → `На уточнении` → `Одобрена` / `Отклонена`

### Покупка места (только для торговцев)

- Интерактивная карта помещения с выбором конкретного стола
- Временная блокировка выбранного места на **10 минут** при открытии карточки
- Поддержка промокодов
- Оплата через подключённый платёжный шлюз
- После оплаты — QR-код аккредитации с номером места

### Расписание (только для музыкантов)

- Слот назначается организатором в admin-панели
- Музыкант видит: дату, время, сцену, технический райдер
- Уведомление при изменении слота

### Загрузка фото (только для фотографов)

- Загрузка фотографий по альбомам
- Отправка на модерацию
- Статусы фото: `На модерации` / `Опубликовано` / `Отклонено`
- При отклонении — причина и возможность загрузить замену

### Чаты с организаторами

Раздельные каналы по категориям:

| Канал | Участники |
|-------|-----------|
| Канал торговцев | Торговцы + организаторы |
| Канал музыкантов | Музыканты + организаторы |
| Канал косплееров | Косплееры + организаторы |
| Канал фотографов | Фотографы + организаторы |

Прямая связь с организаторами доступна только верифицированным пользователям.
