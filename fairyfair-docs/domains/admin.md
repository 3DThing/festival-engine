# admin.fairyfair.ru — Панель администрирования

Закрытый поддомен для команды организаторов. Доступ только через VPN или именной сертификат + обязательная двухфакторная аутентификация.

> **Важно:** Панель физически изолирована от публичной сети. Даже при обнаружении уязвимости на fairyfair.ru вектор атаки на admin не открывается автоматически.

---

## Роли

Роли настраиваются IT-администратором. Набор доступных модулей определяется ролью.

| Роль | Доступ |
|------|--------|
| **IT-администратор** | Управление системой, настройки, роли, VPN-сертификаты |
| **Организатор** | Доступ ко всем модулям |
| **Соорганизатор** | Доступ ко всем модулям |
| **Бухгалтер** | Финансовые операции |
| **Контентмейкер** | Новостная лента, фотоальбомы, модерация |
| **Волонтёр** | Чат поддержки, модерация чатов |
| **Охрана** | Только Android-приложение верификации |

## Модули

### Дашборд

Панель метрик в реальном времени:

- Количество проданных билетов
- Количество занятых торговых мест
- Количество пришедших посетителей (по сканированиям QR)
- Прогноз посещаемости (по купленным билетам)
- Лента аудита действий команды
- Уведомления о критических событиях

### CRM

Система управления пользователями, разделённая на сегменты:

- **Посетители** — зарегистрированные и анонимные покупатели
- **Торговцы**
- **Музыканты**
- **Косплееры**
- **Фотографы**

Возможности CRM:

- Массовая рассылка с сегментацией по категории и параметрам
- Расчёт ценности клиента
- Управление учётными данными пользователя
- Назначение административных ролей с автовыдачей VPN-доступа или сертификата

> Анонимные покупатели (без аккаунта) отображаются как записи с email и ФИО. Рассылка для них — только по email.

### Билеты

- Список всех купленных билетов с привязкой к пользователю
- Поиск по ФИО, email, номеру заказа
- История сканирований по каждому билету
- Отзыв билета и инициирование возврата
- Возврат вручную бухгалтером при ошибке шлюза
- Настройка стоимости билета
- Настройка срока возврата (количество дней до мероприятия)

### Карта

CAD-редактор схемы помещения:

- Расстановка столов и торговых мест перетаскиванием
- Поддержка нескольких этажей / зон
- Установка цены за конкретное место или зону
- Пометка технических зон как недоступных
- Режим черновика (торговцы не видят до публикации)
- Синхронизация с панелью торговцев (занятые места отображаются в реальном времени)

> Купленное место нельзя удалить — доступна только заморозка с уведомлением торговца.

### Финансы

- Доходы: билеты, аренда мест
- Расходы по категориям с комментариями (аренда площадки, техника, и др.)
- Ручное добавление доходов и расходов
- Корректировка баланса с обязательным комментарием
- Автоматический расчёт: выручка, доход, маржинальность
- История изменений по каждой записи
- Экспорт отчёта

### Инвентаризация

Система учёта материалов и оборудования:

- Категории: инструменты, столы, колонки, подсветка и др.
- Автогенерация порядкового номера для каждой позиции
- Указание места хранения
- Статусы: на складе / выдано / утрачено
- Комментарий при пометке как утраченного

### Почтовый клиент

Почтовый клиент для домена `@fairyfair.ru`:

- Общая почта, доступная всей команде
- Личные почтовые ящики сотрудников
- Подключение внешнего почтового ящика (требуется указать почтовый сервер)
- Уведомления о новых письмах

### Чат с организаторами (поддержка)

- Список тикетов от посетителей
- Назначение тикета на оператора
- Опциональный AI-бот с настраиваемой базой знаний
- Эскалация к живому оператору
- Уведомления о новых тикетах в Telegram

### Чаты (опционально)

Система логирования и хранения переписок по каналам:

| Канал | Участники |
|-------|-----------|
| Посетители | Чат посетителей сайта |
| Торговцы | Чат торговцев с организаторами |
| Музыканты | Чат музыкантов с организаторами |
| Косплееры | Чат косплееров с организаторами |
| Фотографы | Чат фотографов с организаторами |
| Волонтёры | Внутренний канал команды |

- Выгрузка любого чата в формате JSON

### Настройки

Модуль конфигурации системы (доступен IT-администратору):

| Раздел | Содержимое |
|--------|-----------|
| Домены | Настройка привязки доменов |
| Сервер | Параметры сервера |
| Ключи и API | Ключи шифрования, токены интеграций (VK, Telegram, платёжные шлюзы) |
| Бэкап | Ручной и автоматический бэкап, восстановление из точки |
| Модули | Включение и выключение модулей системы |
| Роли | Настройка набора прав для каждой роли |
