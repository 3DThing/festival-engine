# fairyfair.ru — Публичный сайт

Основной домен платформы. Пользовательская часть, доступная всем желающим без ограничений.

## Функционал

### Регистрация и авторизация

- Регистрация по email и паролю
- Авторизация через VK OAuth2
- Восстановление пароля по email
- Покупка билета без регистрации (только ФИО и email)

### Покупка билета

- Покупка без регистрации: ФИО + email, билет приходит на почту с номером заказа
- Покупка с аккаунтом: билет сохраняется в личном кабинете, доступны промокоды
- Поиск билета без аккаунта по номеру заказа + email
- Возврат билета в установленный период (срок настраивается в системе)

### Лента новостей

- Публикации от организаторов: анонсы, обновления, расписание
- Доступна без авторизации в режиме чтения

### Фотоальбом

- Альбомы с фотографиями с мероприятий
- Фото загружаются аккредитованными фотографами, публикуются после модерации

### Чат с организаторами

- Система тикетов поддержки
- Опциональный AI-ассистент с базой знаний
- Эскалация к живому оператору
- Уведомления о новых ответах

### Чат посетителей

- Открытый чат для общения участников между собой
- Модерация волонтёрами и контентмейкерами

### Отзывы о фестивале

- Доступны всем купившим билет — с аккаунтом и без
- Без аккаунта: верификация по номеру заказа + email
- Публикуются после модерации организатором

## Личный кабинет (для зарегистрированных)

- Список купленных билетов с QR-кодами
- История заказов
- Управление данными профиля (имя, email, пароль)
- Самостоятельный возврат билета
