# Обзор проекта

## Проблематика

### Отсутствие внешней площадки

Платформа ВКонтакте теряет органический охват — новые алгоритмы препятствуют продвижению мероприятий. Организаторам требуется независимый внешний сайт.

Существующие решения закрывают задачу лишь частично:

| Платформа | Ограничения |
|-----------|-------------|
| Tilda | Ограниченный функционал, невозможность интеграции кастомных модулей |
| WordPress | Сложность интеграции кастомных модулей, высокий порог кастомизации |

### Фрагментация инструментов

Для эффективного управления мероприятием организатору сегодня требуется набор отдельных сервисов:

- Система управления финансами
- CRM (управление посетителями и участниками)
- Аналитика
- Инструменты маркетинга и рассылок
- Поддержка пользователей

Каждый инструмент требует отдельной подписки или ведения локального учёта. Единой системы, объединяющей весь этот функционал, не существует.

### Законодательные ограничения РФ

**Федеральный закон № 115-ФЗ** «О противодействии легализации доходов...»
- Блокирует переводы с карты на карту для приёма оплаты за участие
- Риски блокировки счетов как у отправителя, так и у получателя
- Требует использования легальных платёжных шлюзов (ЮКасса, Тинькофф и др.)

**Федеральный закон № 152-ФЗ** «О персональных данных»
- Обязательное хранение данных пользователей на серверах на территории РФ
- Логирование действий пользователей в течение установленного периода
- Обеспечение технической безопасности хранения персональных данных

---

## Цель

Создание единой платформы управления мероприятиями, устраняющей все описанные проблемы: собственный сайт с гибким функционалом, встроенный набор инструментов администрирования, легальный приём оплаты, соответствие 152-ФЗ.

## Архитектура платформы

```
┌─────────────────────────────────────────────────────┐
│                  Публичная зона                      │
│                                                     │
│   fairyfair.ru          staff.fairyfair.ru          │
│   (посетители)          (участники)                 │
└─────────────────────────┬───────────────────────────┘
                          │
┌─────────────────────────▼───────────────────────────┐
│              Закрытая зона (VPN + 2FA)               │
│                                                     │
│              admin.fairyfair.ru                     │
│              (команда организаторов)                │
└─────────────────────────────────────────────────────┘
                          │
┌─────────────────────────▼───────────────────────────┐
│                Android-приложение                    │
│                (охрана, верификация QR)              │
└─────────────────────────────────────────────────────┘
```

### Домены

| Домен | Назначение | Доступ |
|-------|-----------|--------|
| `fairyfair.ru` | Публичный сайт — посетители | Открытый |
| `staff.fairyfair.ru` | Панель участников — торговцы, музыканты, косплееры, фотографы | Регистрация + аккредитация |
| `admin.fairyfair.ru` | Панель администрирования | VPN / сертификат + 2FA |
| Android App | Верификация посетителей по QR | Токен от IT-администратора |

## Сторонние интеграции

| Интеграция | Назначение |
|------------|-----------|
| VK API | Публикация анонсов, работа с аудиторией |
| VK OAuth2 | Авторизация пользователей через ВКонтакте |
| Telegram API | Уведомления команды, оповещения о критических событиях |
| ЮКасса / Тинькофф / другие | Приём оплаты за билеты и аренду мест |
