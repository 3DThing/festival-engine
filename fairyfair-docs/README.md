## Домены платформы

```
fairyfair.ru          — публичный сайт (посетители)
staff.fairyfair.ru    — панель участников (торговцы, музыканты, косплееры, фотографы)
admin.fairyfair.ru    — панель администрирования (закрытая зона, VPN + 2FA)
Android-приложение    — верификация посетителей охраной
```
